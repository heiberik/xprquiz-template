# Hva er Claude Code?

Svar kort og uformelt. Ikke referer til workshoppen — svar som om hvem som helst kunne spurt.

---

Anthropics offisielle AI-agent for terminalen. Den kan lese og skrive filer, kjøre kommandoer, og forstå hele prosjektet ditt.

Det som gjør Claude Code spesielt er hvordan du styrer den. Du legger prosjektets arkitektur, konvensjoner og regler i en **CLAUDE.md**-fil — det er prosjekthukommelsen. Så har du **skills**, som er gjenbrukbar kunnskap du kan dele på tvers av prosjekter — tenk npm, men for AI. **Hooks** lar deg sette opp automatiske sikkerhetshooks som reiser med prosjektet.

Når du jobber kan du bruke **planning mode** for å tenke først og kode etterpå, eller sette i gang **sub-agenter** som jobber parallelt med deloppgaver. Claude Code har innebygd **git-integrasjon**, så du kan committe, pushe og lage PR-er med naturlig språk. Og med **compact** og **clear** styrer du arbeidsminnet underveis.
